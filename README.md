# Diamond Dynamics Tool

Diamond Dynamics is a game-by-game performance analysis tool built to evaluate how player performance evolves over time.

Rather than focusing on final averages, this tool measures how performance behaves within a season.

## Features

- Consistency Score: Measures stability across games
- Adjustment Score: Measures deviation from recent performance
- Game-by-game trend visualization
- Interactive metric selection

## How It Works

Upload a game-by-game dataset and select a performance metric (OPS, OBP, etc.).

The tool evaluates each game in the context of prior performance, allowing users to identify trends, stability, and shifts in performance over time.

## Tech Stack

- Python
- Streamlit
- Pandas
- Altair

## Purpose

This project is part of the Diamond Dynamics framework, focused on understanding performance as a dynamic system rather than a static outcome.
